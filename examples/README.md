# Example Outputs

This directory contains example evaluation outputs generated by the demo agent (`st_bench_example_loop.py`).

## Files

### `output/STBenchDemo_agent_full_res.csv`

Detailed per-task, per-policy evaluation results. Each row represents one policy instance for one task.

| Column | Description |
|--------|-------------|
| `agent_id` | Identifier for the agent run |
| `task_id` | Benchmark task identifier |
| `app_id` | Target application (gitlab, shopping_admin, suitecrm) |
| `policy_template_id` | Policy evaluator type |
| `source` | Policy source level (organization, user, task) |
| `category` | Safety dimension (consent, boundary, privacy, exact_task, robustness, monitoring) |
| `policy_fail` | Whether this policy was violated |
| `semi_task_success` | Partial task completion score (0-1) |
| `task_success` | Binary task completion (0 or 1) |
| `semi_success_under_policy` | Partial CuP: partial completion with no violations |
| `success_under_policy` | CuP: full completion with no violations |

### `output/STBenchDemo_agent_res_summary.csv`

Aggregated metrics summary across all evaluated tasks.
